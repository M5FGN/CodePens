# Weight Convertor

## Description

Create a weight conversion tool to convert lbs to kgs and vice versa.

## MVP

* Accepts an input value.
* Accepts an input unit.
* Accepts an output value.
* Calculates the output value based on the above information.  

## Technologies Used

* HTML
* CSS
* JavaScript
